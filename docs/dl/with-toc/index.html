    Activation functions #

    Sigmoid #

    \[ \sigma(z)=\frac{1}{1+\exp{(-x)}} \]

    Usage: add non-linearity to linear outputs s.t. \(\sigma(z)\in(0,1)\) and \(\sigma(0)=0.5\) .

    Tanh #

    \[ \tanh(z)=\frac{\exp{(x)}-\exp{(-x)}}{\exp{(x)}+\exp{(-x)}} \]

    Usage: add non-linearity to linear outputs s.t. \(\tanh(z)\in(-1,1)\) and \(\tanh(0)=0\) .

    ReLU * #

    \[ \text{ReLU}(x)=\max{(0,x)} \]

    Name: Rectified Linear Unit

    Usage: convert negative linear outputs to 0.

    LeakyReLU #

    \[ \text{LeakyReLU}(x)=\max{(\alpha x,x)} \]
    • \(\alpha\in(0,1)\) : hyperparameter (negative slope), default 0.01.

    Name: Leaky Rectified Linear Unit

    Usage: scale negative linear outputs by \(\alpha\) .

    PReLU * #

    \[ \text{PReLU}(x)=\max{(\alpha x,x)} \]
    • \(\alpha\in(0,1)\) : learnable parameter (negative slope), default 0.25.

    Name: Parametric Rectified Linear Unit

    Usage: scale negative linear outputs by a learnable \(\alpha\)

    ELU * #

    \[ \text{ELU}(x)=\begin{cases} x & \text{if}\ x\geq0 \\ \alpha(\exp{(x)}-1) & \text{if}\ x<0 \end{cases} \]
    • \(\alpha\) : hyperparameter, default 1.

    Name: Exponential Linear Unit

    Usage: convert negative linear outputs to the non-linear function above.

    GELU * #

    \[ \text{GELU}(x)=x*\Phi(x) \]

    Name: Gaussian Error Linear Unit

    Usage: weigh each output value by its Gaussian cdf.

    Softmax * #

    \[ \text{softmax}(x_i)=\frac{\exp{(x_i)}}{\sum_j{\exp{(x_j)}}} \]

    Usage: convert each value in the output tensor into its corresponding exponential probability s.t. \(\sum_i{\text{softmax}(x_i)}=1\) .
